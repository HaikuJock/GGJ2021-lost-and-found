## Project & Team

This was a project for the 2021 Global Game Jam. Project and Team can be found [here](https://globalgamejam.org/2021/games/little-bit-lost-prince-1).

## Local Development

Install npm:

https://nodejs.org/en/

Mac: Install live-server:

`npm install -g live-server`

Mac: Start server in project root:

`live-server`

Windows: Install http-server:

`npm install -g http-server`

Windows: Start server in project root:

`http-server -c-1`

Go to:
 
http://localhost:8080

Mac: Server will automatically reload browser on changes.

## Deploy

Push your changes and the project will deploy automatically to:

https://andersmurphy.github.io/GGJ2021-lost-and-found/
